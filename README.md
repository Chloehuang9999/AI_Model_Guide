# AI Model Guide

静态网页部署目录，主文件为 `index.html`。

## 在线访问

- **Vercel 部署**：<https://ai-model-guide.vercel.app>

## 本地预览

```bash
npm install
npm run dev
```

打开 `http://localhost:4173`。

## 部署方式

- **Vercel**：已包含 `vercel.json`，导入该目录即可部署。
- **Netlify**：已包含 `netlify.toml`，发布目录为当前目录。
- **任意静态托管**：直接上传整个目录内容（保留 `assets/`）。
